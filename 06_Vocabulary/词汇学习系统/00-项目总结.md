## 💡 高级玩法

### 1. 自定义提示词
编辑 `AI词汇生成器.py`,添加您的特殊要求:
```python
prompt = f"""
额外要求:
- 例句必须来自剑桥雅思真题
- 搭配必须是COCA高频搭配
- 记忆技巧要有趣且实用
"""
```

### 2. 批量处理优化
```python
# 分批处理,避免API限流
for i in range(0, len(word_list), 20):
    batch = word_list[i:i+20]
    generator.batch_generate(batch)
    time.sleep(5)
```

### 3. 缓存机制
避免重复生成,节省成本:
```python
# 检查缓存
cached = get_cached_note(word)
if cached:
    return cached
else:
    # 生成并缓存
    note = generator.generate(word)
    save_to_cache(word, note)
```

### 4. 多主题批量生成
```python
themes = ['教育', '环境', '科技']
for theme in themes:
    word_list = get_words_by_theme(theme)
    generator.batch_generate(word_list, theme=theme)
```
