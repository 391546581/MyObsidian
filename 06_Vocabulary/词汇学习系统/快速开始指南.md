subtitle_folder = "path/to/subtitles"
for file in os.listdir(subtitle_folder):

## 🆘 常见问题

### Q1: Obsidian SR插件和Anki哪个更好?

**答**: 
- **SR插件**: 适合日常学习,与Obsidian无缝集成,数据本地化
- **Anki**: 适合考前冲刺,移动端体验更好,算法更成熟

**建议**: 日常用SR插件,考前1个月切换到Anki强化

---

### Q2: 如何从字幕中提取具体例句?

**答**: 修改Python脚本,添加例句提取功能:

```python
def extract_sentences_with_word(self, text, word):
    """提取包含指定单词的句子"""
    sentences = re.split(r'[.!?]+', text)
    return [s.strip() for s in sentences if word in s.lower()]
```

---

### Q3: 词汇太多,复习不过来怎么办?

**答**: 
1. **优先级排序**: Brand7 > Brand9 > Brand5
2. **减少新词**: 每天新增5-10个,而非20+
3. **延长间隔**: SR插件中调整"Easy"按钮的间隔
4. **删除低频词**: 删除出现频率<3的词汇

---

### Q4: 如何与雅思真题结合?

**答**:
1. 做完真题后,标记不认识的词
2. 查询词汇是否在您的库中
3. 如果不在,按Brand分级添加
4. 在笔记中标注"来源: 剑15 Test 1"
