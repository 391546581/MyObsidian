## 🔄 完整工作流示例

### 场景: 从学习视频到雅思应用

```
第1天: 输入
├─ 观看英语学习视频 (30分钟)
├─ 下载字幕文件 (.srt)
└─ 运行: python 词汇提取工具.py
   └─ 输出: 50个高频词汇

第2天: 处理
├─ 运行: python AI词汇生成器.py --batch
├─ AI生成50个完整笔记 (5分钟)
└─ 人工审核和微调 (10分钟)

第3-30天: 复习
├─ 每天Obsidian SR复习 (10分钟)
├─ Brand5: 100%掌握
├─ Brand7: 80%掌握
└─ Brand9: 60%掌握

第31天: 应用
├─ 雅思写作Task2练习
├─ 使用Brand7词汇替换基础词
├─ 词汇分数从6.0提升到7.0
└─ 总分提升0.5-1.0分
```
