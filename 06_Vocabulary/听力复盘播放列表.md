# 🎧 听力复盘列表: {{date}}

```dataview
TABLE without id 
("![[" + file.name + "]]") as "音频播放",
file.link as "来源笔记",
file.cday as "创建时间"
FROM "06_Vocabulary/Assets"
WHERE file.extension = "webm" OR file.extension = "mp3"
SORT file.ctime DESC
LIMIT 20
```

> **使用说明**:
> 1. 此页面使用 Dataview 插件自动提取 `06_Vocabulary/Assets` 下的所有录音文件。
> 2. 实现“连续播放”：Obsidian 原生不支持自动连播，建议安装 **"Media Extended"** 插件，或手动点击播放。
> 3. 复习策略：点击音频听发音 -> 回想含义 -> 点击“来源笔记”查看详情。
