Trancy 和沉浸式翻译 两个插件的区别，谁更好用呢

Trancy 与沉浸式翻译的核心区别在于定位：Trancy 是 “视频优先 + 语言学习” 工具，沉浸式翻译是 “全场景翻译 + 效率优先” 工具Immersive Translate。日常翻译选沉浸式翻译更全能稳定；看视频学语言选 Trancy 更贴合学习需求Immersive Translate。

### 二、关键功能对比（可直接复用）

#### 1. 视频场景

- Trancy：剧院模式 + 阅读模式，支持影子跟读、填空练习、AI 语法分析，适合学语言。
- 沉浸式翻译：双语字幕 + 排版优化，无专项学习功能，适合快速理解内容Immersive Translate。

#### 2. 网页翻译

- Trancy：基础双语对照，样式可定制，侧重配合视频场景。
- 沉浸式翻译：智能识别主内容，排版优化，支持 20 + 翻译引擎，适配 Google/Twitter 等多网站Immersive Translate。

#### 3. 文档翻译

- Trancy：仅支持基础字幕 / 文本翻译，无复杂文档能力。
- 沉浸式翻译：PDF（含 OCR）、EPUB、Word 等，保留排版，适合论文 / 合同 / 技术文档Immersive Translate。

#### 4. 学习功能

- Trancy：跟读 / 填空 / 生词本 / AI 语法分析，适合系统学语言。
- 沉浸式翻译：无专项学习功能，专注翻译效率Immersive Translate。

### 二、选型建议（按场景）

1. 语言学习（看视频练听力 / 口语）：选 Trancy，专项学习功能是核心优势。
2. 日常网页 / PDF / 多格式文档翻译：选沉浸式翻译，全场景覆盖 + 排版优化 + 多引擎更高效Immersive Translate。
3. 兼顾视频与文档：优先沉浸式翻译，必要时搭配 Trancy 做视频学习Immersive Translate。
4. 预算有限：两者免费版足够日常；Trancy 会员解锁学习进阶，沉浸式翻译会员解锁 OCR / 批量 PDFImmersive Translate。

---

### 三、快速上手与注意事项

1. Trancy：Chrome/Edge 商店安装，打开 YouTube→启用插件→选 “练习模式” 开始学习。
2. 沉浸式翻译：安装后 Alt+A 翻译网页，Alt+W 翻译整页，支持自定义翻译引擎与快捷键沉浸式翻译。
3. 注意：Trancy 对部分小众视频站适配可能不稳；沉浸式翻译在复杂 PDF/OCR 场景需会员Immersive Translate。
