# 🛠️ 商业分析画布 (SWOT)

## 🛠️ 用途
- **商业决策**: 分析新产品上市、公司战略转型。
- **个人发展**: 职业规划、转行分析。
- **竞品分析**: 我方与敌方的优劣势对比。

## 📖 使用指南 (SOP)
1. [ ] **明确目标**: 确定你要分析的具体对象（如：是否应该在这个月启动新项目？）。
2. [ ] **环境扫描**: 
   - 内部：盘点资源、团队、核心竞争力（找优势/劣势）。
   - 外部：看政策、市场趋势、竞争对手动态（找机会/威胁）。
3. [ ] **填空画布**: 使用 [[Template-SWOT-Canvas]] 模板进行头脑风暴。
4. [ ] **制定策略**: 重点关注交叉分析（如 SO/WT），产出至少 3 个具体的 Todo。
